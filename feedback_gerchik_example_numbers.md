При извлечении правил из курса А.М. Герчика: конкретные числа, которые он называет в РАЗБОРЕ
примера (напр. «проторговка 90-100, средняя позиция крупняка 95»), — это иллюстрация механики,
а НЕ параметр для хардкода.

**Why:** Артём поправил 2026-06-10 — «те цифры взяты для примера, в формулу их зашивать не нужно».
Уже была близкая ошибка с порогами (см. [[feedback_trading_analysis_protocol]]).

**How to apply:** кодировать ОБЩИЙ метод, а пример использовать только для проверки. Для проторговки —
VWAP зоны (90-100→95 это просто VWAP равномерной зоны, а не магия числа 95). Отличать «правило с числом»
(напр. стоп=10% ATR, паранормал>2×ATR — реальные пороги) от «число в примере». При сомнении — общий
расчёт + порог как настраиваемая константа, не литерал из разбора. Связано: [[project_gerchik_strategy_extraction]].
