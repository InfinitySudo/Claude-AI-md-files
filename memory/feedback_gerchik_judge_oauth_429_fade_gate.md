2026-06-13: серия из 6 шортов TAOUSDT подряд, ВСЕ в SL. TAO в этот день +25% (O212→C266) —
агент серийно ШОРТИЛ ПАРАБОЛУ, фейдя каждый уровень 233→268 на пути вверх.

**Корень (торговля):** не было жёсткого гейта против контртренд-фейда. Гейты `chasing_exhausted` и
`no_atr_room` ловят только ход В СТОРОНУ сделки; фейд ПРОТИВ сильного свежего хода (шорт в ракету)
проходил насквозь. `against_trend` лишь ×0.5 стоп (делал хуже — выбивало быстрее). RS (stronger_than_market)
— только edge-флаг, не гейт.
**Fix:** `FADE_MAX_COUNTER_ATR` (config `fade_max_counter_atr=1.0`) в pipeline: для фейдов (отбой/ЛП)
если день сделал ≥1×ATR ПРОТИВ сделки и туда закрылся (exh_dir против side) → skip `countertrend_runaway`.
Симметрично chasing. Живьём сразу сработал (WLDUSDT). Это и есть «научить не шортить ракету».

**Корень (судья/деньги):**
- Vision-судья на ANTHROPIC_API_KEY → 400 `credit balance too low`: ключ Артёма БЕЗ кредитов
  (сожрал ~$5 пользы 0). Удалён из .env по требованию Артёма.
- Баг: `claude_auth.get_client()` создавал `Anthropic(auth_token=...)` БЕЗ `api_key=None` → SDK
  подхватывал ANTHROPIC_API_KEY из env и слал ЕГО вместо OAuth → 400 даже при выбранном OAuth.
  Fix: `api_key=None` явно в get_client.
- OAuth (подписка) отдаёт **429 стабильно** на программные вызовы (3/3, не транзиент) — серверный
  лимит Anthropic на API-использование подписочного токена, НЕ зависит от интерактивного юза Артёма;
  усугублён шарингом токена между агентами VPS (тьюторы/OpenClaw). Отменить нельзя.
- **Решение:** `use_vision=false` → rule-судья (детерминированный, бесплатный, мгновенный). Торговлю
  не теряем: решения принимают rule-гейты Герчика + фейд-гейт. Vision вернуть = пополнить API-ключ.

**Урок:** качество входа чинится на УРОВНЕ ПРАВИЛ (фейд-гейт), а не Vision-судьёй; судья был «толку 0».
Зелёные уровни на чарте у TAO = supports НИЖЕ текущей цены (цена ушла вверх после шортов) — на момент
входа они были resistance над/у цены. См. [[feedback_tutor_oauth_auth_token]] [[project_gerchik_instrument_selection]].
