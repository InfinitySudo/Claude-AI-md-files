В OnTime (/root/ontime, домен ontime.management) Артём разрешает действовать без подтверждений по любым правкам — UI, бэкенд, цвета, схемы, рестарты сервиса, билды, деплой.

**Why:** Артём не хочет ловить меня вопросами на каждой текучке; уже отдельно настраивали `feedback_no_routine_confirms.md` для других проектов, в OnTime тоже подтвердил явно (2026-04-15).

**How to apply:** Делать сразу, без "хочешь, я сделаю?". Единственное исключение — удаление объектов (DELETE /api/projects/{id} или сноска db/таблиц/файлов с пользовательскими данными). Для удаления — спросить.
