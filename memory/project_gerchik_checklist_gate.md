Каждый стиль (пробой/отбой/ЛП) = **чек-лист предпосылок из СЫРЫХ транскриптов** (`/root/gerchik_course/*/*.txt`, не из сводок — они переакцентировали). Гейт: **выполнено ≥50% предпосылок стиля → вход**, иначе skip `checklist_below_50`. `edge_score = checklist_frac` (доля выполненных). Чек-лист ✓/✗ + N/M виден в описании сделки (таблица/попап/открытые позиции). commit 1e302a6.

**4 кита трейдинга** (подтв. дословно урок `[b7f60d3]`): уровень (главный) / энергия (накопление-контртренд-новости) / направление / точка входа.

**Реализация:** `src/gerchik/checklist.py` — `build_checklist(strategy, ctx)` + детекторы. Пайплайн собирает ctx из готовых флагов + новых детекторов, `evaluate()` → гейт. `PreThesis.checklist`/`checklist_frac`. config `checklist_min` (0.5).

**Новые детекторы (урок `[757c3cc2]`, раньше НЕ было):** закрытие дневки далеко/близко от уровня в ATR (далеко→отбой/ЛП, близко→пробой); открытие ≤50% ATR до уровня (пробой не брать); «не пустили ≥3 раз» `[0478da60]`; зона заражённости за уровнем `[757c3cc2][03c021c7]`.
