**Контекст:** Toptal закрыт ([[project_toptal_waitlist]]). RU-SMB agency — bridge-канал из [[project_10k_strategy]] №2: $2-5k setup + $500-1500/мес support, до первого контракта 3-6 нед при активной работе.

**Why этот канал первый:**
- Артём говорит на русском, иммигрантский network в Calgary + Toronto + Vancouver уже есть через TSA, борцовский комьюнити, OnTime customers.
- SMB-владельцы из СНГ готовы платить за «нашего» подрядчика быстрее, чем за cold US-агентство.
- Не требует портфолио на en-LinkedIn; кейсы (OnTime/TSA, Wrestling Tracker, voice-tutor) уже на русском говорят сами.

## ICP (кого ловим)

**Профиль:**
- Русскоязычный собственник SMB в Canada/US.
- Бизнес: construction/siding/roofing, e-commerce, logistics, кафе/сервис, авто, страхование/риелторство.
- 5-50 сотрудников, $500k-5M годовой оборот.
- Болит: ручной learnорот в Excel/WhatsApp, потеря заявок, отсутствие учёта, формен пишет отчёты на коленке.

**Где они:**
- TG-каналы / чаты: «Русские в Калгари», «РусТоронто», «Бизнес США-Канада», профильные чаты siding/construction.
- Facebook-группы иммигрантских комьюнити.
- Сарафан через TSA / борцовский клуб / Constant Wrestling parents.

---

## Оффер (что продаём)

**Не «AI-агентство», а конкретные продукты-пакеты:**

1. **Field Ops Tracker** (на базе OnTime) — $1500 setup + $300/мес.  
   Учёт рабочих часов, проектов, материалов, отчётов с фото для бригад. Кейс: TSA уже работает.

2. **CRM-автоматизация** — $2000-3000 setup + $500/мес.  
   Telegram-бот для приёма заявок → Notion/Airtable/Google Sheets, авто-уведомления, шаблоны ответов.

3. **AI voice/chat-ассистент** — $3000-5000 setup + $800-1500/мес.  
   На базе voice-tutor стека: бот отвечает клиентам в WhatsApp/TG голосом или текстом, фильтрует мусор, передаёт горячих в CRM. Кейс: AISmartFriendBot, wife-english-tutor.

4. **Custom AI-интеграция** — почасовка $80-120/ч или fixed $2-5k.  
   Если у клиента уже есть Salesforce/HubSpot/QuickBooks — встроить Claude/AI-workflow.

**Якорь продажи:** «Я делаю это для TSA (40 человек в поле, $X экономии в месяц). Покажу скриншоты.»

## Каналы привлечения

**Активные (push):**
1. **Личные сообщения в TG-чатах** — не спам, а целевые контакты после комментариев в темах про «как считать часы / как принимать заявки». 5-10 сообщений в день.
2. **Знакомые из TSA + Constant Wrestling** — попросить интро к 3-5 владельцам бизнеса в комьюнити.
3. **Пост в @solo_claude** ([[project_tg_channel_plan]]) с кейсом OnTime — конверсия в DM возможна.

**Пассивные (pull, делать параллельно):**
4. **Лендинг ontime.management/business** или отдельный домен — описание пакетов на русском.
5. **2-3 статьи на Habr про реальные кейсы** (OnTime для siding, voice-bot для customer service).

> «Здравствуйте, [Имя]. Видел ваш комментарий в [чат] про [боль]. У меня есть рабочее решение — делал для siding-компании в Калгари. Учёт часов, отчёты с фото, бюджет по проектам. 40 человек на проде уже год. Если интересно — могу за 10 минут показать как это могло бы выглядеть у вас. Бесплатно, без обязательств.»

**Цель первого звонка:** не продать, а получить **3 конкретные боли** + согласие на демо. Демо = 30 мин по Zoom через 2-3 дня.

## Ценник — psychology

- **Не «$50/час»** — это junior-floor. SMB-владелец слышит «дешёвый разработчик».
- **«$1500 setup + $300/мес»** — это «продукт», и звучит как SaaS-подписка, к которой они привыкли.
- Скидку давать только за **3-мес предоплату** ($800 вместо $900) или за **кейс с правом публикации**.

## Чего НЕ делать

- Не подписываться на «весь стек под ключ за $500» — съест время от OnTime.
- Не работать без 50% предоплаты (русскоязычные SMB любят торговаться задним числом).
- Не брать клиентов вне русскоязычного сегмента в первые 90 дней — там длиннее цикл продажи.
- Не путать с OnTime SaaS-моделью: agency = bespoke; OnTime = self-serve. Не каннибализировать.
